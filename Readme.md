# Namaste React
This is the tutorial of React

In this lecture we have ignited our app(created our own create-react-app) <br>
step1: npm init<br>
step2: npm install -D parcel  (-D is dev dependency)<br>
step3: we removed the cdn links of react and react-dom from index.html and make the script tag type module of javascript tag App.js <br>
step4: we have install react and react-dom - npm install react and npm install react-dom .<br>
step5: now we can start our parcel server by writing- npm parcel index.html <br>
step6: we can add the browsers list in package.json - we can tell our code which version of browsers it should run on. <br>
step7: when we start our parcel server two folder have created these are dist and .parcel-cache these two folders are help parcel to build our code fast by caching and then it use HMR(hot module replacement) to render on browser in no time. <b>we have to add these two folders in .gitignore file because these folder will automatically created when we will start our server.</b><br>
